# Portfolio_Frontend-
Description
This project is a frontend application built with React. The application allows users to create an account, contact the owner, and chat directly within the platform. The purpose of this project is to provide an easy way for users to communicate and interact.

Features
User Authentication: Users can create an account, log in, and log out.
Contact Form: Users can reach out to the owner directly via a contact form.
Live Chat: Users can chat with the owner in real-time.
Responsive Design: The application is fully responsive and works well on all devices.
Multi-Language Support: The application supports multiple languages.
Tech Stack
Frontend: React, React Router, Axios
Styling: Tailwind CSS
State Management: React Context API
Backend :Laravel
Build Tool: Vite.js
